How does it work?

At the core of Project ORKA is an always evolving, community driven, modular code base, accessible to everyone.

What is it for?

Every game built with Project ORKA will be 100% moddable, meaning that games can always be improved and evolve even if the original developer ceases to work on it.

What is the goal?

Project ORKA is not meant for making cash or ripping off gamers like the big studios do, Project ORKA is for the people who love making, modding and playing games.

Who is it for?

People that have seemingly simple ideas and want that executed as fast as possible to see where it leads, whithout having to write a whole game from scratch.

What comes next?

At the core games are just software and that means that project ORKA not only applies to games but all kinds of software.

We want to create a passionate community take modding to the next level boost game development make coding effortless
